### Run the App Locally

1. Create a virtual environment:

- On macOS and Linux:

```bash
python3 -m venv venv
```

- On Windows:

```bash
python -m venv venv
```

2. Activate the virtual environment:

- On macOS and Linux:

```bash
source venv/bin/activate
```

- On Windows:

```bash
venv\Scripts\activate
```

3. Install the dependencies:

- On macOS and Linux:

```bash
pip3 install -r requirements.txt
```

- On Windows:

```bash
pip install -r requirements.txt
```

4. Run the app:

- On macOS and Linux:

```bash
python3 app.py
```

- On Windows:

```bash
python app.py
```
